Adjusting Entity Boxes

Getting into shape

By default images added to an entity will take on the position and size of the entity's rectangle. The box around an entity is determined by it's position and size components. We could fix this by changing the size of the entity to match the image size, or add properties to the image component to make it a specific size. To keep things simple let's change the size of the player entity like this:


{
"id": 1,
"name": "player",
...
"size": {
	 "width": 32,
	 "height": 48
 },
...
}

Now save again and the player should be the correct size, and not stretched.
